# GuestrixAI (Archive)

GuestrixAI (Archive) is a sanitized archive of a real-time conversational AI platform for Airbnb guest communications built with Flask, Socket.IO, and Google Gemini. It captures the core architecture, tooling, and product workflows from the original system in a portfolio-friendly form.

## Main Features
- Real-time text and voice guest conversations with AI routing and tool calls
- Host and guest web dashboards with reservation context
- Knowledge base ingestion and retrieval with vector search
- Airbnb property data import and enrichment workflows
- Firebase phone authentication and magic link access patterns
- Deployment tooling for Flask and the Telnyx voice WebSocket service

## Repository Overview
- `concierge/` — Flask web app, APIs, dashboards, and backend utilities
- `websocket/` — Telnyx + Gemini Live bidirectional voice service
- `docs_current/` — Curated project documentation preserved in this archive

## Disclaimer
This codebase is a sanitized archive of a dissolved startup, presented here for non-commercial portfolio demonstration purposes.
